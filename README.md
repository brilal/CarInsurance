# CarInsurance

ASP.NET MVC project for calculating car insurance quotes.  
Includes quote logic, admin view, and user form for creating insuree records.

## Features
- Calculates monthly insurance quotes based on age, car details, DUI, and coverage type.
- Displays all quotes in an Admin view.
- Uses Entity Framework for database access.

## Technologies
- C#
- ASP.NET MVC
- Entity Framework
- Visual Studio
